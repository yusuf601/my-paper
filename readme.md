# Guide to Download the Paper

Follow the steps below to download the paper from the GitHub repository:

1. Open the **Actions** tab in the repository.
2. Select the **latest workflow run**.
3. Scroll down to the **Artifacts** or **Upload PDF** section.
4. Look for an item containing text similar to **`Artifact download URL`**.
5. Click the provided link.
6. The **PDF paper** will be downloaded to your computer.
